1️⃣  The Key Tasks to Complete in the Dashboard

Our work centers around a few key tasks using the left-hand navigation bar.  Get to know these areas the best.

Calendar - Events

#Channels - group messaging similar to a Slack group

Segments & Campaigns - messaging groups of students all at once

Peer Connections - DM'ing individual students

Programs - all of the learning content

Your first task is to get in and start focusing on seeing how these sections are set up. You will see many other things in here - steer clear of Emails, Invitations, Commerce or sales-y type buttons - those are used by the sales team and are connected to other systems that make it a little more complicated.

If you run into an issue, Slack me so we can look into it.  The system has many intuitive features, but if you need support, the Audit logs (bottom) can track history so we can always go back and reset if needed.    ​
