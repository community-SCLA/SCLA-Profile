COURSE DESCRIPTION

OKR Mastery: Strategy in Action is a leadership and performance course that teaches students how to set clear goals, track progress, and execute plans using the Objectives and Key Results (OKR) framework. OKRs are widely used in modern organizations to connect vision, strategy, and daily work, helping individuals and teams stay focused on what matters most.

This course explains how to create strong objectives, define measurable results, and use structured planning to improve performance. Students will learn how clear goals increase motivation, reduce confusion, and make it easier to measure success. The course also explores how OKRs can be applied to personal growth, academic work, career planning, and leadership development.

Through guided exercises, planning sessions, and real-world examples, students will practice writing OKRs, tracking progress, and adjusting strategy when goals change. The course emphasizes discipline, clarity, and accountability as essential skills for high performance.

By the end of the course, students will be able to design and use OKRs to manage projects, improve productivity, and turn ideas into action.

COURSE OBJECTIVES

Students will:

• Understand the OKR framework
• Learn the difference between goals and results
• Write clear objectives
• Create measurable key results
• Track progress effectively
• Adjust plans when needed
• Improve strategic thinking
• Apply OKRs to school and career
• Build accountability habits
• Create a personal strategy system

Course work requires approximately 4–6 hours per week for 12 weeks.
Assignments must be completed with a score of 80% or higher.

COURSE TIMELINE / OUTLINE

Capstone Project:
Students will create and present a Personal OKR Strategy Plan that includes objectives, key results, tracking methods, and a system for reviewing progress in academic, personal, and professional goals.
