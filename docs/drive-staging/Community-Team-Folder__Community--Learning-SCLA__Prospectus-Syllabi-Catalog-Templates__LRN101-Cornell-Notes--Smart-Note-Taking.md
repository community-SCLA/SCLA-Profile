COURSE DESCRIPTION

Cornell Notes: Smart Note-Taking is a learning skills course designed to teach students how to use the Cornell Note-Taking System to improve understanding, memory, and academic performance. Effective note-taking is one of the most important skills for success in school, career training, and professional development, yet many students have never been taught a structured method.

In this course, students will learn the Cornell Notes framework, which helps organize information, identify key ideas, and review material efficiently. Students will practice taking notes from lectures, readings, videos, and discussions while learning how to summarize, ask questions, and retain important information.

The course also focuses on active learning, study strategies, and review techniques that help students understand material more deeply instead of memorizing without meaning. Students will use Cornell Notes across different subjects and apply the method to real academic and professional situations.

By the end of the course, students will be able to use the Cornell Note-Taking system independently to improve learning, organization, and performance.

COURSE OBJECTIVES

Students will:

• Understand the purpose of structured note-taking
• Learn the Cornell Notes format
• Practice note-taking from lectures and readings
• Improve listening and focus skills
• Identify key ideas and supporting details
• Write effective summaries
• Use notes for studying and review
• Develop active learning habits
• Improve organization skills
• Create a personal note-taking system

Course work requires approximately 4–6 hours per week for 12 weeks.
Assignments must be completed with a score of 80% or higher.

COURSE TIMELINE / OUTLINE
