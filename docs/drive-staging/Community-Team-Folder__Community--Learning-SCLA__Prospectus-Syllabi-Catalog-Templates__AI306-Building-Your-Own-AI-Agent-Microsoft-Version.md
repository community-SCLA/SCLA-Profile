# AI306: Building Your Own AI Agent (Microsoft Version)

COURSE DESCRIPTION

Building Your Own AI Agent (Microsoft Version) is a hands-on course designed to teach students how to create, customize, and use AI agents to improve productivity, automate tasks, and support professional decision-making. As artificial intelligence becomes integrated into modern workplaces, the ability to build and manage AI tools is an essential career skill.

This course focuses on using Microsoft AI technologies and platforms to design personal and professional AI agents. Students will learn how to structure prompts, create workflows, connect tools, and develop AI assistants that can complete real-world tasks such as writing, research, scheduling, analysis, and communication.

Through guided projects, step-by-step demonstrations, and practical exercises, students will build their own AI agent and learn how to use it responsibly, efficiently, and ethically.

By the end of the course, students will have a working AI agent and a repeatable system for using AI to increase performance in academic, business, and leadership settings.

COURSE OBJECTIVES

Students will:

• Understand the concept of AI agents
• Learn the basics of Microsoft AI tools
• Build prompts for structured workflows
• Create automated task systems
• Connect AI to productivity tools
• Use AI for writing, research, and planning
• Improve efficiency using AI assistants
• Apply ethical and responsible AI use
• Design a personal AI workflow
• Build a functional AI agent

Course work requires approximately 4–6 hours per week for 12 weeks.
Assignments must be completed with a score of 80% or higher.

COURSE TIMELINE / OUTLINE

Capstone Project:
Students will design, build, and demonstrate a personal AI agent using Microsoft tools. The project must show how the agent improves productivity, organization, and professional performance.
