# `content/` — content creation projects

Newsletters, social posts, blog drafts, and content series.

## What belongs here

- Newsletter issues (drafts and finals)
- Social media copy batches
- Blog posts and long-form content
- Content calendars or series planning docs

## Key sources to reference

- `scla/source-of-truth/voice-decisions.md` — locked brand voice constraints
- `scla/brand/voice-and-tone.md` — tone guidance per channel
- `scla/source-of-truth/program-names.md` — canonical program names to use in copy

## Status lifecycle

`draft` → `active` (in editing) → `complete` (published or delivered)
