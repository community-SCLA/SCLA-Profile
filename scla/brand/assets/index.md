## Logo system overview

SCLA's logo system has three components:

- **Wordmark** (`SCLA-Logo.svg`) — the "SCLA" text lockup; primary logo for most contexts
- **Icon mark** (`SCLA-logo-icon.svg`) — compact oval-ring icon in gold; use when space is limited or a square format is required
- **Shape mark** (`logo-shape.svg`) — the abstract concentric-oval graphic in SCLA blue; decorative, used as a background element or standalone graphic accent

Both the icon and shape mark derive from the same oval/ring motif — the icon is a tight 5-ring stack in gold; the shape is a wider layered form in blue.
