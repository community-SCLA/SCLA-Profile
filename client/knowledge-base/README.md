# Knowledge Base

Generated by the `knowledge-architect` agent from `client/_raw/docs/` and
`client/_raw/web/`. Run `/kb` to (re)build.

## Files

- `index.md` — entry point + "start here" for new readers
- `glossary.md` — acronyms, code-names, internal terms
- `people-and-teams.md` — who does what
- `products-and-services.md` — what the company sells
- `systems-and-tools.md` — software stack + ownership
- `faqs.md` — recurring questions from `_raw/artifacts/`

Topic pages (e.g. `customers.md`, `compliance.md`) may be added as warranted.

## Contract

See `.claude/agents/knowledge-architect.md` for the full rules:
- Quotes over paraphrases
- Every claim links back to `_raw/`
- One entity → one canonical name
- Gaps become `TODO: needs input`, never fabrications

## Regeneration

This directory is **fully regenerated** every time `/kb` runs. Hand-edits
will be lost. If you want to preserve changes, move them into
`client/source-of-truth/` (which is never regenerated after first pass).
