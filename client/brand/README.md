# Brand

Generated by the `brand-analyst` agent from `client/_raw/web/` and
`client/_raw/assets/`. Run `/brand` to (re)build.

## Files

- `brand-guide.md` — top-level index + "common mistakes"
- `voice-and-tone.md` — how we write, with real copy samples
- `visual-identity.md` — colors, typography, logos, design tokens
- `assets/index.md` — inventory of every logo/image with usage rules

## Design tokens

`visual-identity.md` includes both a YAML block and a CSS `:root { ... }`
block suitable for dropping into a codebase.

## Regeneration

This directory is **fully regenerated** every time `/brand` runs. Hand-edits
will be lost. Preserve refinements by moving them into
`client/source-of-truth/` or by updating the source material in `_raw/`.
