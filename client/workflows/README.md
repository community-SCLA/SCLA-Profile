# Workflows

Generated by the `workflow-mapper` agent from `client/_raw/artifacts/`. Run
`/workflows` to (re)build.

## Files

- `current-state.md` — one section per recurring workflow (trigger, steps, handoffs, pain points)
- `automation-opportunities.md` — prioritized table (P0/P1/P2, S/M/L/XL effort)
- `ship-fast-playbook.md` — top 5 opinionated recommendations

## How opportunities are found

The `workflow-analyzer` skill scans artifacts for four patterns:

| Signal | Suggests |
|---|---|
| Same text copy-pasted between tools | Integration |
| Human decisions that correlate ≥90% with a feature | Rule/bot |
| "Wait, what was decided?" in threads | Context-passing doc |
| Meetings with ≥80% status updates | Written summary |

## Regeneration

Overwritten every time `/workflows` runs.
