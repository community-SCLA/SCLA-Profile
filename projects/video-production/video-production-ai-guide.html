What "30 Hours" Actually Means

  • 30 hours = 1,800 minutes of finished video per peak month
  • At 44 videos per course × 2–7 min each = 88–308 minutes per course
  • Social: 22 videos × 60–90 sec = 22–33 minutes
  • Company Tour: 8 videos × 8 min each = 64 minutes
🚨
Critical Scale Reality: Synthesia's Creator plan allows 30 min/month. HeyGen's Creator plan allows ~10 min of premium Avatar IV video per month. At your 1,800 min/month target, you need Enterprise tier on whichever platform handles course video bulk. Contact both vendors about nonprofit/education pricing before committing to a pipeline build.

PART 4 Step-by-Step Workflows

A
Course / Certificate Video (AI Actor + B-Roll)
Target: 5–15 min training videos · Claude → Synthesia → Wistia
⏱ 30–60 min human effort per video (after first template setup)
Claude (script) → Synthesia template → Auto render → Human QA → Wistia upload

Step 1 — Script Generation (Claude)

I'm creating a training video for an SCLA course called [Course Name].
The audience is college students preparing for [topic].
The video should be [X] minutes long, structured as:
1. Hook / Problem Statement
2. Key Concept Explanation
3. Practical Example or Story
4. Key Takeaways
5. Call to Action

Tone: Warm, professional, encouraging. SCLA voice.
Format: Write each scene as a separate paragraph. Include [B-ROLL: description]
markers where motion graphics would appear.
Keep each paragraph to ~30 seconds of speaking time.

Topic: [paste your course content or outline here]

Human review required before proceeding.

Step 2 — Build / Reuse Synthesia Template

First time only: Login → New Video → Choose Template → set SCLA brand colors (#eaab2d, #3393d6) → select avatar → save as Brand Kit.

Each new video: Open saved template → Duplicate → paste script into scene fields → place B-roll markers.

Step 3 — Generate Video AUTOMATED

Click Generate. Expected render time: ~5–15 min per finished minute. Receive email notification when done.

Step 4 — QA Review HUMAN REQUIRED

  • Script accuracy — no hallucinated facts
  • Avatar lip sync quality throughout
  • B-roll timing appropriate
  • SCLA branding consistent
  • Audio levels balanced
  • Length matches target

Step 5 — Export and Upload to Wistia

  1. Synthesia: Download MP4 (or SCORM at Enterprise)
  2. Wistia: Upload → auto-generates transcript, captions, AI chapters, embed code
  3. Set access control → embed in course platform
B
Learning Activity / Short Video (AI Actor, <3 min)
Target: Micro-learning, module intros · HeyGen → Wistia
⏱ 15–30 min human effort per video
HeyGen Video Agent → Human QA → Wistia

Option 1 — HeyGen Video Agent (Fastest) FULLY AUTOMATED

Create a 90-second training video for SCLA college students about [topic].
Tone: energetic and professional. Use a diverse, professional-looking avatar.
End with: "Visit your SCLA dashboard to complete the activity."

HeyGen generates script, visuals, avatar, and voiceover automatically. Human QA → approve → download → Wistia.

Option 2 — Script + HeyGen Avatar Studio (More Control)

  1. Generate script in Claude (same structure as Workflow A, shorter)
  2. HeyGen → Avatar Studio → select avatar + voice → paste script → Generate → QA → Wistia
C
Social Media Videos (60–90 sec)
Target: Instagram, TikTok, LinkedIn, YouTube Shorts · HeyGen → Social/Wistia
⏱ 15–20 min human effort per video
  1. Claude → generate 60-second script:
    Write a 60-second social media script for SCLA's [platform] audience. Topic: [X]. Hook in first 3 seconds. End with a clear CTA. Short punchy sentences. One idea per line.
  2. HeyGen → 700+ templates → "Social" category → select vertical (9:16) or square (1:1)
  3. Paste script → select avatar → generate
  4. Review → export → post directly or upload to Wistia for analytics tracking
💡
Batch tip: HeyGen's Batch Mode lets you generate multiple avatar/script variations from one CSV. A/B test which avatar performs better on each platform.
D
Translation / Multilingual Videos
Target: Any finished video in 175+ languages · HeyGen Video Translation → Wistia
⏱ 5 min human effort + automated render
  1. Start from any finished MP4 (from Synthesia, HeyGen, or traditional production)
  2. HeyGen → Video Translate tab → upload MP4 → select target language(s)
  3. Select mode:
    • Hyperrealistic (recommended for AI avatar videos): Full lip-sync re-animation
    • Audio Dubbing: Voice only, no facial re-animation — works for live-actor footage
  4. Submit → AUTOMATED render (~5 min per minute of video)
  5. Human QA: check tone accuracy and cultural appropriateness
  6. Upload to Wistia with language tag

Bulk Translation (Many Videos at Once)

  1. Prepare a CSV: video_url, target_language_code, output_name
  2. Use HeyGen API bulk endpoint (/v2/video_translate) or Make.com scenario
  3. Batch generates all language variants concurrently (up to 10 simultaneous)
  4. Completed videos auto-upload to Wistia via Zapier
E
Screen Recording Videos
Target: Software tutorials, platform walkthroughs · Screen record → HeyGen voice → Wistia
⚠️
AI can handle the voiceover layer. The screen recording itself is always a manual step — it requires a human to perform and capture real actions.
  1. Record screen using Loom, OBS, or QuickTime (MANUAL — cannot be automated)
  2. Edit recording if needed (cut mistakes, add highlights)
  3. Claude → generate narration script matching the screen actions
  4. HeyGen → Avatar Studio → upload screen recording as background → OR use avatar as picture-in-picture overlay
  5. Voiceover generated from script → synced to video
  6. Export → upload to Wistia
💡
Voice clone tip: If an SCLA team member records 2 minutes of clean audio once, their cloned voice can narrate all future screen recordings automatically — no re-recording needed.
F
Batch Course Production (High Volume)
Target: Full 44-video course library · Claude CSV → Synthesia Batch → Wistia bulk upload

Step 1 — Generate All 44 Scripts at Once (Claude)

I'm creating a complete course with 44 videos for SCLA.
Here is the course outline: [paste outline]

For each section, write a [2-4 min] script:
- Scene 1: Hook + learning objective (30 sec)
- Scene 2: Core concept explanation (60-90 sec)
- Scene 3: Example or application (60-90 sec)
- Scene 4: Summary + transition to next video (30 sec)

Format the output as a CSV with columns:
video_number, video_title, scene_1_script, scene_2_script, scene_3_script, scene_4_script

Use SCLA's warm, professional, encouraging voice throughout.
No markdown in script fields — plain text only.

Step 2 — Bulk Generate in Synthesia AUTOMATED

  1. Synthesia → your course template → Upload script CSV
  2. Map CSV columns to scene slots
  3. Submit batch → Enterprise tier renders all 44 concurrently
  4. Spot-check 5–10 videos for QA (not all 44)

Step 3 — Bulk Upload to Wistia AUTOMATED

  1. Wistia Upload API: use Zapier or script to upload all MP4s at once
  2. Set metadata (title, description) from original CSV
  3. Wistia auto-organizes into the correct channel/folder
  4. Embed codes generated automatically per video

PART 7 One-Time Setup Checklist

Synthesia Setup

  • Log in and verify subscription tier (Enterprise required for bulk volume)
  • Create SCLA Brand Kit: upload logo, set colors (#eaab2d, #3393d6), set fonts
  • Create/select SCLA course avatar (custom from video recording, or stock)
  • Build 3 master templates: Course Video · Certificate Video · Learning Activity
  • Test bulk CSV import with 3 sample scripts → verify output quality
  • Contact Synthesia sales about nonprofit/education pricing

HeyGen Setup

  • Log in and verify subscription tier
  • Record 2-min voice sample for cloning (quiet room, external mic, natural speech)
  • Create cloned voice in HeyGen → Voice Cloning → test output quality
  • Build 3 templates: Short-form Social · Learning Activity · Company Tour narration
  • Configure Instant Avatar (optional — 2-min webcam recording)
  • Test Video Agent with 2–3 topic prompts → evaluate quality
  • Contact HeyGen sales about education/nonprofit pricing

Wistia Setup

  • Confirm plan supports expected storage volume
  • Create Channels: Courses · Certificate Programs · Social · Company Tour
  • Configure folder hierarchy matching content structure
  • Configure player branding (SCLA colors, logo)
  • Connect HubSpot (if applicable) for viewer analytics sync
  • Generate API key for bulk upload scripting
  • Test embed codes in course platform (Canvas/LMS)

Zapier / Make Automation

  • Zap 1: HeyGen video complete → upload MP4 to Wistia folder
  • Zap 2: Wistia upload complete → Slack #video-production notification
  • Zap 3 (optional): Google Sheet row added → trigger HeyGen batch generation
  • Zap 4 (optional): HeyGen translation complete → Wistia with language tag

Cost-Reduction Strategies

  1. Assign tools to lanes — HeyGen for short-form, Synthesia for long-form. Don't duplicate the same video type on both platforms.
  2. Batch render off-peak — avoid 9am–5pm PST for HeyGen large batches; processing is faster and more reliable.
  3. Translate top performers only — use Wistia analytics to find highest-watched videos before investing in translation.
  4. Reuse avatar templates relentlessly — one template setup = zero marginal cost per additional video in that style.
  5. Use AI voiceover over AI avatar for content that doesn't benefit from a talking head (data-heavy, screen recordings) — credits cost less.

PART 10 Known Limitations & Gotchas

Synthesia

  • Minute caps are hard limits — going over on a paid plan causes jobs to queue or fail. Monitor monthly usage.
  • Google Slides requires PPTX export first — no direct connector; add this extra step to your workflow.
  • Interactive videos can't be downloaded as MP4 — choose between interactivity and portability at design time.
  • SCORM export is Enterprise-only — if your LMS requires SCORM, you must be on Enterprise.

HeyGen

  • Renders stall at 33% or 50% — documented platform bug. Resubmit the job if it hangs for more than 30 minutes.
  • Premium credits burn fast — Avatar IV = 20 credits/min. 200-credit Creator plan = only 10 min of Avatar IV video/month.
  • Billing surprises — multiple users report unexpected overages. Set a credit alert threshold in account settings immediately after setup.
  • Long-form consistency drops at 3+ minutes — use Synthesia for training videos longer than 3 minutes.
  • No free API credits as of 2026 — all API usage is paid from the first call.

Wistia

  • Not a native LMS — no SCORM, no gradebook sync. Canvas/Moodle integration is iframe-only.
  • No official nonprofit discount — contact support@wistia.com directly and ask.
  • Viewer-level analytics require CRM integration — without HubSpot, you only see aggregate stats.

Claude

⚠️
Always review AI-generated scripts before production. Claude can be confidently wrong about specific SCLA program details, dates, and statistics. Fact-checking is a required human step — never send a script directly to render without review.

Do not feed FERPA/PII data into Claude for script generation — use anonymized examples and general course content only.
